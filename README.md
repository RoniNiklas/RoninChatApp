# RoninChatApp

# Sovelluksen kuvaus:
Tavoitteena websovellus, jolla käyttäjät voivat viestiä toistensa kanssa reaaliaikaisesti websockettia käyttäen. Teknologioina Node.JS, React, MongoDB, socket.io ja bootstrap.
